# WebSocket Chat System

A learning project to understand design patterns, WebSocket communication, and scalable architecture.

## Quick Start

### Prerequisites
- Java 21
- Docker

### Run the Project

```bash
# Windows
gradlew.bat bootRun

# Linux/Mac
./gradlew bootRun
```

Visit: http://localhost:8080
